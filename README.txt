PDF Bitmap Compare PRO — Руководство по запуску

1. Установить Python 3.13+ (рекомендуется последняя версия).
2. В терминале:
   cd ~/Desktop/PDFBitmapCompare
   pip3 install -r requirements.txt
   python3 main.py

• Иконку hand.png можно заменить на свою (положить в icons/hand.png).
• Переключение языка — в меню сверху.
• Drag&Drop работает, если установлен tkinterdnd2.
• Для macOS: если появляется окно безопасности (Security), разрешите запуск в настройках.
